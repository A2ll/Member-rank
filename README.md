# Member-rank
用户分析平台关键会员层级关系

<img src="/img/层级分析工具.png" width="50%">

1. 首先填写**用户id**和**推荐id**对饮在excel中的列表名
2. 选择Excel文件位置
3. 选择右边功能**数据预处理**
4. 点击**运行**
5. 等待运行完成后，在**分析id**填入要分析的id，再选择**层级分析**得到运行结果。
